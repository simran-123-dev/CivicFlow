# CivicConnect Backend

## Setup

1. Copy .env.example to .env and fill values.
2. Install deps: npm install
3. Run: npm run dev

## Endpoints

- POST /api/auth/register
- POST /api/auth/login
- GET /api/complaints
- POST /api/complaints
- GET /api/complaints/:id
- PATCH /api/complaints/:id
- DELETE /api/complaints/:id
